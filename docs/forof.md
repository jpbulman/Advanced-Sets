# for...of()

### Overview

`AdvancedSet`'s have iterator support, which allows them to be used in `for...of` loops

### Parameters

None

### Usage

```js
const setA = new AdvancedSet(1, 2, 3, 4, 5, 6);
for (const val of setA) {
  console.log(val);
}
```

### Details

Introduced in: 0.0.3
